# Movie Rating Prediction with Python

## Overview
This project predicts movie ratings using machine learning techniques based on features such as genre, director, cast, year of release, and other available attributes. The objective is to analyze historical movie data and build a regression model capable of estimating movie ratings accurately.

## Dataset
- Source:IMDb Movies India.csv
- Target Variable: Movie Rating

## Project Workflow
1. Data loading and exploration
2. Data cleaning and preprocessing
3. Feature engineering and encoding of categorical variables
4. Exploratory data analysis and visualization (including a correlation heatmap)
5. Model training using regression algorithms
6. Model evaluation using metrics such as MAE, RMSE, and R² score

## Results
The final model achieved:
- MAE: 1.049562505489382
- RMSE:1.6864004365428578
- R² Score: 0.08929055052181267
